## Stack

### Front

- Vue 2 - Main framework
- Vuex -  State library
- Vuex ORM 
- TypeScript
- Vue Class Decorators - Vue typescript syntax sugar
- Axios - http client
- Vuetify - component library


### Back

- Express - server
- PostgreSQL - database
- Prisma - ORM (sort of)
- TypeScript
- JWT - Auth
- JOI - Validation
