## Stack

### Front

- Vue 2 - **Front End framework**
- Vuex -  **State library**
- Vuex ORM 
- TypeScript
- Vue Class Decorators -**Vue typescript sugar**
- Axios - **HTTP client**
- Vuetify - **UI library**


### Back

- Express - **HTTP Server**
- PostgreSQL - **Database**
- Prisma - **ORM (sort of)**
- TypeScript
- JWT - **Token generation library**
- JOI - **Validation**
